Introduction

Cupertino Girls Who Code is now a nationally registered club. This program is a year-long project from the lunch club on campus - Women in Tech and it's primary goal is to provide girls exposure to coding and computer science.

Let's close the gender gap in technology, together.

Javascript/Graphics

We work on projects that include javascript to create graphics and animation programs. Using knowledge of javascript, we also develop games that range from basic to complex intensity.

Web Development

During our bi-weekly meetings, we learn the basics of HTML and CSS to create websites and display information. Girl Coders from beginning to expert skills are all welcome!

National Curriculumn

We use the national resources provided by girls who code. These resources allow for Tino Girl Coders to learn from a variety of lessons and activities.
